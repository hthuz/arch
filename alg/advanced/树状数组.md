# 树状数组

Binary Indexed Tree (BIT)



对于 tree[i], 其维护 [i - lowbit(i) + 1, i]这个区间的和

比如

| i    | lowbit(i) | interval |
| ---- | --------- | -------- |
| 1    | 1         | [1,1]    |
| 2    | 2         | [1,2]    |
| 3    | 1         | [3,3]    |
| 4    | 4         | [1,4]    |
| 5    | 1         | [4,5]    |
| 6    | 2         | [5,6]    |
| 7    | 1         | [7,7]    |
| 8    | 8         | [1,8]    |

type BIT struct {
    n int
    tree []int
